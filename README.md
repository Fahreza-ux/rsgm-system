# 🦷 Sistem Informasi Manajemen RSGM (Rumah Sakit Gigi & Mulut)

Sistem informasi berbasis web yang dirancang untuk mengelola administrasi rumah sakit gigi, mulai dari proses login staf, pemantauan antrean pasien, hingga manajemen rekam medis secara efisien dan responsif.

## 🚀 Demo
Anda dapat melihat tampilan live aplikasi ini melalui GitHub Pages:
[Link Demo Anda di Sini] (Contoh: https://username.github.io/rsgm-system/)

## ✨ Fitur Utama
* **Modern UI/UX**: Menggunakan Tailwind CSS untuk tampilan yang bersih dan profesional.
* **Responsive Design**: Optimal untuk diakses melalui perangkat desktop maupun mobile.
* **Dashboard Statistik**: Ringkasan jumlah pasien, antrean, dan dokter yang bertugas secara real-time.
* **Manajemen Antrean**: Tabel status pasien (Menunggu/Selesai) yang mudah dipahami.
* **Secure-look Login**: Halaman autentikasi untuk keamanan akses staf medis.

## 🛠️ Teknologi yang Digunakan
* **HTML5**: Struktur semantik website.
* **Tailwind CSS**: Framework CSS untuk styling cepat dan modern.
* **Font Awesome**: Library ikon untuk antarmuka yang intuitif.
* **JavaScript (Vanilla)**: Logika sederhana untuk navigasi dan interaksi UI.
